История
Changelog

  • Медиа запросов

    Исправлены медиа запросы, некоретное отображение сгенерированного изображения при печати, изображение растягивалось и не центрировалось в диалогом окне печати

  • Кнопка "История"

    Добавлена новая кнопка меню, при нажатии на кнопку открывается котекстное меню "История"

  • Контекстное меню "История"

    Контекстное меню, которое содержит историю созданых QR-кодов за текущую сессию

  • Стили

    Были исправлена стилицая отоброжения некоторых элементов, так же исправлены медиа запросы и медиа print запросы

  • функционал открытия QR-кода из истории

    При нажатии на QR-код в истории открывает новую страницы about:blank с ранее генерируем QR-Кодом

  • Кнопка "Печать Qr code"

    Изменение функционала кнопки,в предыдущих версиях кнопка печатала QR на домашней страницы, в последущих версиях кнопка открывает новую страницы about:blank

  • Оптимизация частиц

    Значительные просадки FPS из-за долго нахождения на странице. в предыдущих версиях каждая частица генерировалась в индевидуальном теге div, в последующих версиях используется тэг <canvas>

  • Исправление meta тэгов

    Улучшение стилей, изменение структуры DOM, подготовка к переходу с .css на .scss

  • Автоматическое закрытие окна печати

    После печати QR-кода открытая страница about:blank закрывается автоматически, через 2 секунды

  • Индикатор времени

    Добавлен индикатор времени перед автоматическим закрытием окна печати

  • Кнопка "Changelog"

    Добавлена новая кнопка меню, при нажатии на кнопку открывается котекстное меню "Changelog"

  • Контекстное меню "Changelog"

    Контекстное меню, которое содержит историю обновлений Веб страницы

  • Кнопка "Удалить пробелы"

    При нажатии на кнопку удаляет все лишние пробелы из поля ввода текста для генерации QR-кода"

  • Переход на SCSS

    Полностью переписанная стилизация веб страницы, код переписан с чистого .css на препроцессор .scss !

  • Media Requests

    Полностью переписаны и оптимизированны все Media Requests с .css на препроцессор .scss, старый файл media.css удалён из проэкта

  • Перписаны функции отрисовки частиц

    Переписаны функции отрисовки частиц, в новой версии установлен лимит в 300 частиц, при достижении количества частиц данного значения. Предыдущие версии не учитывали накопление частиц а так же не успевали удалять достаточное количество частиц чтобы избежать их накопление

  • Предварительный просмотр Changelog

    Изменение стилизации предпросмотра Changelog

  • Генерация QR-Кода

    Визуальное изменение в генерации QR-Кода. Ранее при генерации QR-Кода происходило смещение из-за центрирования элементов в Контейнере. В последующих версиях происходит плавная генерация.

    Так же был добавлена анимация загрузки QR-Кода

  • Удалить пробелы

    Удалена кнопка "Удалить пробелы"

  • Кнопки "Открыть все" "Закрыть все" в Changelog

    Добавлены кнопки для открытия всего списка изменений по версиям приложения в Changelog

  • Checkbox "Удалить лишние пробелы"

    Добавлен input типа checkbox с двумя состояниями, в состоянии checked(состояние по умолчанию) автоматически убирает все лишние пробелы из поля ввода и гененрирует QR-Код без пробелов, в состоянии unchecked позволяет ввод в поле ввода пробелы, символы переноса, абзацы и пустые символы

  • Checkbox "Повреждённый заказ"

    Добавлен input типа checkbox с двумя состояниями, в состоянии checked добавляет подпись "ПОВРЕЖДЁННЫЙ ЗАКАЗ" к генерации QR-Код, в состоянии unchecked(состояние по умолчанию) подпись "ПОВРЕЖДЁННЫЙ ЗАКАЗ" отсутствует

  • Сочетание клавиш Ctrl+Z

    При Сочетание клавтшь Ctrl+Z очищаются поля ввода генерации QR-Кода, а так же сбрасывается текущий QR-Код
